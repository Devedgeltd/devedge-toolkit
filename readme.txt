A growing collection of useful WordPress admin enhancements by DevEdge Ltd. Includes search by post ID, post ID columns in admin, comment post ID display and search, and GitHub-based update support.

== Description ==
DevEdge Toolkit makes WordPress admin management easier by adding:
- Search by post ID functionality
- Display of post IDs in post/page/custom post type listings
- Display of post IDs in the comments table
- Search comments by post ID
- Future features and auto-updates via GitHub releases

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory or install via WordPress plugin upload.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings > DevEdge Toolkit to enable/disable features.
